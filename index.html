My name is Xianxu Hou, now I am a graduate student studying Geology (till June, 2014) in China University of Mining & Technology, Beijing. At the same time, I am a computer hobbyist.

Though I take some introduction courses about computer languages during my undergraduate study, I really start to learn programming since I began studying in graduate school.

One of my best friend who is now a software engineer recommended that SICP is exactly the book I should read. Thus, with this amazing book, I embarked on my journey of programming. Additionally I took the corresponding online course provided by MIT OpenCourseWare.

I was so impressed by the beauty of coding such as functional programming after SICP, that I decide to learn more in detail. I began learn Racket with online book HTDP, at the same time I began to write code with DrRacket

After that, I took the Introduction to Computer Science and Programming course on MIT OpenCourseWare. Meanwhile I learned python which is both simple and powerful, and started to solve problems on Codeforces.

I am so lucky that MOOC such as Coursera, EdX appears recent years, which gives a wonderful apportunity to learn programming from the best universities around the world. I learned algorithm from Algorithms: Design and Analysis by Prof. Tim Roughgarden . Though it is a challenge for me, I have learned a lot and have fun in this class. What's more, based on Startup Engineering by Prof. Balaji Srinivasan and some related books and documents, I start to learn the basic operation of Linux(Ubuntu) system, set up an HTML page, do some basic design, style it in CSS and version control the code in git. Finally use node.js and express framework to set up webapp, then deploy my application with heroku.
